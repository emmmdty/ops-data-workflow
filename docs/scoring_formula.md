# 投流效益评分公式

本文记录 `analysis.xlsx`、`total_summary.xlsx` 和 Dashboard 中使用的评分口径。当前实现位置在 `ops_data_workflow/pipeline.py` 的 `_add_scoring_columns`。

## 适用范围

评分用于两类汇总表：

- `分渠道栏目题材排名`：按 `渠道 + 账号 + 二级栏目 + 三级题材` 汇总后评分。
- `内容类型分级表` / 内容榜单：按 `二级栏目 + 三级题材` 汇总后评分。

当前口径不启用一级分类，不按长视频、短视频、图文做一级维度；渠道是第一层分析维度。

评分是同一张汇总表内部的相对排名分，不是跨周期绝对分。新增或删除行会改变百分位排名，因此也会改变评分。

## 基础指标

每个汇总行先计算：

- `点击率 = 点击量 / 展示曝光量`
- `激活成本 = 消耗 / 激活数`
- `首次付费成本 = 消耗 / 首次付费次数`
- `首次付费率 = 首次付费次数 / 激活数`

如果分母为空或为 0，对应派生率/成本留空。参与评分时，空值按 0 处理后计算百分位排名。

## 百分位排名

所有评分都基于 `rank(pct=True, method="average")`：

- 数值越大，百分位越高。
- 并列值取平均百分位。
- 空值先按 0 处理。

记作：

```text
P(x) = x 在当前汇总表中的百分位排名
```

激活成本是成本指标，越低越好，因此使用反向百分位：

```text
低成本分 = 1 - P(激活成本)
```

## 热度评分

热度评分衡量内容获得曝光和点击的能力：

```text
热度评分 =
  0.45 * P(展示曝光量)
  + 0.35 * P(点击量)
  + 0.20 * P(点击率)
```

权重含义：

- 展示曝光量 45%：衡量内容获得流量的能力。
- 点击量 35%：衡量内容实际吸引点击的能力。
- 点击率 20%：衡量曝光转点击效率。

## 拉新评分

拉新评分衡量内容带来激活和首次付费的综合能力：

```text
拉新评分 =
  0.35 * P(激活数)
  + 0.25 * P(首次付费次数)
  + 0.15 * (1 - P(激活成本))
  + 0.15 * P(首次付费率)
  + 0.10 * 热度评分
```

权重含义：

- 激活数 35%：核心拉新规模。
- 首次付费次数 25%：深层转化规模。
- 激活成本 15%：越低越好，用反向百分位。
- 首次付费率 15%：激活后的付费效率。
- 热度评分 10%：给兼具流量潜力的内容加权。

## 拉新综合评分

拉新综合评分是页面和报表默认排序指标：

```text
拉新综合评分 = round(拉新评分 * 100, 2)
```

默认排序：

- `分渠道栏目题材排名`：先按渠道分组，再按拉新综合评分从高到低排序。
- 内容榜单：按拉新综合评分从高到低排序。

## 注意事项

- 评分只用于同一批次、同一汇总表内比较，不建议直接作为跨周期绝对分。
- 当汇总行数很少时，百分位评分会更粗糙，应结合消耗、激活数、首次付费次数和成本一起判断。
- 数据缺失会被折算为 0 参与百分位排名，因此缺失点击、激活或付费数据会拉低相关评分。
