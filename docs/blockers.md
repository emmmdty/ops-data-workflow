# 项目阻塞记录

## 账号名自动补全

- **B站账号名**：当前原始表只有 `Up主mid`，这是 B站用户的数字 MID/UID，不是账号名。已确认 `1622777305` 对应账号名为 `同花顺投资`，系统会自动映射。其他未知 MID 仍无法仅凭原始表确认账号名；公开请求 B站空间/API 可能返回风控/非法访问页面，无法在无 Cookie、无人工验证的环境中稳定解析。当前处理：已知 MID 自动补账号名，未知 MID 保存在 `account_id`，`account` 和 `author` 留空，等待后续提供账号映射表或带账号名的导出字段。
- **抖音账号名**：当前样例表未包含 `账号`、`账号名称`、`发布账号`、`达人名称` 等账号名字段，也未包含可稳定换取账号名的账号 ID 字段。当前处理：字段不足时 `account` 留空；后续导出增加账号字段后自动读取。
- **小红书账号名**：当前样例表包含 `发布作者`，可以直接作为账号名和作者名使用；暂无阻塞。

## 处理原则

- 不把平台预期账号、平台名称或数字 ID 猜测为账号名。
- 账号名只来自原始导出字段、人工审核映射表，或无需登录且可稳定访问的公开资料。
- 遇到接口风控、登录要求、验证码或字段缺失时，账号名留空，并在统一清洗 workbook 的诊断 sheet 或重点内容审核队列中体现缺失。
